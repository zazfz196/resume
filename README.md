# 📄 Currículo - Roma Miguel

Um currículo web responsivo e moderno, desenvolvido com HTML5, CSS3 e JavaScript vanilla.

## 🎯 Sobre

Projeto de portfólio pessoal apresentando experiência profissional, habilidades técnicas e formação acadêmica. Site totalmente responsivo e otimizado para visualização em qualquer dispositivo.

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com Flexbox
- **JavaScript** - Animações interativas
- **Responsive Design** - Mobile-first

## 📋 Seções

- **Header** - Nome, cargo e informações de contato
- **Objetivo** - Descrição profissional
- **Experiência** - Histórico profissional com datas e responsabilidades
- **Habilidades** - Tags com competências técnicas
- **Formação** - Educação e certificações
- **Footer** - Informações adicionais

## 📱 Features

✅ Design limpo e profissional  
✅ Totalmente responsivo  
✅ Animações suaves ao carregar  
✅ Sem dependências externas  
✅ Pronto para imprimir como PDF  

## 🚀 Como Usar

1. Clone ou baixe este repositório
2. Abra `index.html` em seu navegador
3. Personalize com suas informações
4. Imprima como PDF (Ctrl+P) para gerar currículo em PDF
